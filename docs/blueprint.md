# **App Name**: TaskHome

## Core Features:

- Task Listing: Display available tasks with descriptions, requirements, and rewards.
- Task Submission: Allow users to submit proof of task completion.
- Payment Processing: Manage user balances and process payments for completed tasks.
- User Authentication: Secure user accounts with login and registration functionality.
- Progressive Web App (PWA) Installation: Enable users to install the website as a PWA for offline access and app-like experience.
- Task Recommendation: Use AI to learn which tasks are a good match for each user; then prioritize showing these in their task list. Provide the LLM with the list of the user's past submissions as a tool.
- AI Task Approval Assistant: Utilize AI to review completed task submissions, detect fraud, and suggest approval or rejection. The task will be approved by the poster, admin or dev. For each submission, give the LLM tool access to data on the history of user behavior, task completion speed, typical success rate, reported fraud from the task issuer, and associated user ratings to assist the task approval.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey trust and professionalism.
- Background color: Light gray (#F0F0F0), a desaturated near-white hue, for a clean, modern backdrop.
- Accent color: Violet (#8A2BE2), an analogous color, for call-to-action elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, offering a neutral and modern appearance, making it versatile for both headings and body text.
- Use clear, modern icons to represent different task categories.
- A clean, card-based layout to display tasks and user information effectively.
- Subtle transitions and loading animations to improve user experience.
